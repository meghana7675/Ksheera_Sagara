# 📌 Ksheera Sagara – Android App

Ksheera Sagara is a dairy management Android application designed to help farmers digitally manage milk production, quality analysis (Fat & SNF), and automated payment calculations. It replaces manual milk slips and improves accuracy, transparency, and efficiency in dairy operations.

---

## 🚀 Features
- Daily milk entry management system  
- Automatic milk payment calculation based on Fat & SNF  
- Expense tracking module  
- Income and production summary reports  
- Digital replacement of manual milk slip system  
- Simple and user-friendly interface  
- Secure local data storage using database  

---

## 🛠️ Technologies Used
- Android Studio  
- Kotlin / Java  
- XML (UI Design)  
- SQLite Database  
- Room Database  
- Jetpack Compose  
- MVVM Architecture  
- ViewModel & LiveData  
- Coroutines (for background tasks)  
- Material Design Components  
- Git & GitHub (Version control)  

---

## 🎯 Objective
To digitize dairy farm management and reduce manual calculation errors by providing an automated, transparent, and efficient digital solution for milk collection, quality tracking, and payment processing.

---

## 🏗️ System Architecture
The application follows **MVVM (Model-View-ViewModel)** architecture:
- **Model:** Handles data using Room/SQLite  
- **View:** UI built using XML / Jetpack Compose  
- **ViewModel:** Manages UI-related data and business logic  

---

## 📱 Modules
