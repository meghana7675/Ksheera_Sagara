# 📌 Ksheera Sagara – Android App

Ksheera Sagara is a dairy management Android application that helps farmers manage milk production, quality (Fat & SNF), and payments digitally. It replaces manual milk slips and improves accuracy and transparency.

---

## 🚀 Features
- Daily milk entry management  
- Automatic payment calculation based on Fat & SNF  
- Expense tracking system  
- Summary reports of income and production  
- Simple and user-friendly UI  

---

## 🛠️ Tech Stack
- Android Studio  
- Kotlin / Java  
- XML  
- SQLite  
- Room database
- jetpack compose
